## Чеклист безопасности (перед продакшеном)

### Секреты
- Все секреты находятся только в `.env`.
- В репозитории нет `BOT_TOKEN`, паролей, ключей (проверка grep/сканером).

### SQL-инъекции
- Все обращения к БД выполняются через SQLAlchemy ORM или параметризованные запросы.
- Входные данные валидируются (Pydantic).

### Логирование
- Логи имеют уровни DEBUG/INFO/WARNING/ERROR.
- Настроена ротация и ограничение суммарного размера логов **<= 100MB**.

### Зависимости и уязвимости
- `pip-audit` без критических проблем.
- `npm audit` без критических проблем (или с documented mitigation).
- Удалены лишние зависимости.

### Производительность
- Нет тяжёлых вычислений на запрос.
- Проверен путь “ход -> ответ” (минимальные задержки).

### Админка
- Доступ только после авторизации.
- Rate limit на логин.
- Secure cookies (HttpOnly/SameSite).
- (Опционально) включена “маскировка” через `ADMIN_ROUTE_SECRET`.
