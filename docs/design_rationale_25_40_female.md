## Обоснование дизайна под ЦА: женщины 25–40

### Цель
Сделать игру визуально приятной, “спокойной” и современной, чтобы она воспринималась как небольшой расслабляющий ритуал, а не как “олдскульная” аркада.

### Наблюдения по ЦА (обобщённо)
- Ценится **эстетика и аккуратность**: мягкие цвета, чистая типографика, удобные отступы.
- Важно **не утомлять**: минимум визуального шума, умеренные анимации.
- Позитивный тон и поддерживающие формулировки повышают вовлечённость.

### Решения по UI
**Цвета**
- Базовая палитра: кремовый/пудровый/лавандовый + тёмный текст для контраста.
- Акценты: мягкий градиент и “светящиеся” состояния выбранных клеток.

**Типографика**
- Хорошо читаемый шрифт для основного текста.
- Лёгкий акцентный шрифт для заголовков/промокода.

**Анимации**
- Быстрые и мягкие (150–250ms).
- При победе: лёгкий эффект радости (конфетти/мерцание), без агрессии.

**Копирайтинг**
- Вместо “Вы проиграли” использовать более мягкие варианты вроде “Сегодня победил компьютер — сыграем ещё?” (но без инфантильности).

### Настраиваемость из админки
- Интенсивность анимаций.
- Основные цвета темы.
- Тексты сообщений (в рамках шаблонов).
