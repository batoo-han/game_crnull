## Frontend (Web)

Браузерный интерфейс игры и скрытой админки.

### Цель дизайна
UI адаптирован под женскую аудиторию 25–40 лет: спокойная “премиальная” палитра, мягкие анимации, чистая типографика.

### Запуск (локально)
Подробная инструкция: `docs/development_windows_venv.md`.

Коротко:
1) Установка:

```bash
cd apps/web
npm install --no-fund --no-audit
```

2) Для dev добавьте `apps/web/.env.local`:

```text
VITE_API_BASE=http://localhost:8000
```

3) Запуск:

```bash
npm run dev
```

По умолчанию dev-сервер Vite слушает порт `5174` (жёстко, чтобы не было “случайных” портов).
